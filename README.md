# 点击烟花效果 H5 页面

这是一个简单的HTML5页面，当用户点击屏幕任意位置时，会在点击处绽放彩色烟花效果。

## 功能特性

- 点击任意位置触发烟花爆炸效果
- 多彩烟花粒子向四周扩散
- 支持鼠标点击和触摸屏操作
- 响应式设计，适配各种设备
- 纯前端实现，无需额外依赖

## 技术实现

- HTML5 + CSS3 + JavaScript
- 使用Web Animations API实现平滑动画效果
- 随机颜色生成器为每次烟花提供不同色彩
- CSS动画控制粒子运动轨迹

## 使用方法

1. 打开页面
2. 点击屏幕任意位置
3. 观赏烟花效果
